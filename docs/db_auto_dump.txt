Выполняет ежедневный дамп БД.
Временные файлы лежат в /tmp/vn_dumps/
Всегда хранится один отчет - предпоследний
Скрипт настроен на запуск кроном под пользователем root (необходимо для команды дампа)
Скрипт auto_db_dump_vn.sh расположен в главном каталоге проекта visual_novel
В качестве почтового демона используется mailx

Для настроек почты использовать команду mcedit /etc/mail.rc
